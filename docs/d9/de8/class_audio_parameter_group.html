Detailed Description

Manages and organizes a group of audio parameters by type.

The AudioParameterGroup class allows for the efficient management and organization of audio parameters by grouping them together. This class supports various types of parameters, such as sliders, buttons, and choices, and provides easy methods for adding, retrieving, and managing these parameters within the group.
