Detailed Description

A class representing a button UI element with various states and actions.

This class can read inputs from the GUI and Bela Analog Input. All incoming button values should be of type momentary, where:

  • 0 indicates the button is pushed
  • 1 indicates the button is not pushed

Call update() periodically to ensure proper functioning. The analog value will be debounced.

If a new value is detected, the class determines the type of action: CLICK, LONGPRESS, or RELEASE, and notifies its listeners with the corresponding message.
