Detailed Description

A stereo filter class for audio processing, implementing Moog ladder and half-ladder filter models.

The FilterStereo class processes stereo audio samples through either a Moog ladder or a half-ladder filter model. It allows the user to adjust the cutoff frequency, resonance, and filter model for various audio filtering effects.
