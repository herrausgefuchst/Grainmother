Detailed Description

Caches display content, allowing it to be formatted and retrieved as needed.

The DisplayCache struct stores messages and values that are to be displayed. It allows adding different types of data (e.g., float, int, string), creating rows for display, and clearing or printing the cache. The cache can be used to update the display across different platforms.
