Detailed Description

A transposed direct form 1st-order filter for stereo audio processing.

The TPT1stOrderFilterStereo class implements a 1st-order lowpass (LPF) or allpass (APF) filter for processing stereo audio samples. It uses a transposed direct form to provide efficient filtering.
