void Reverb::setReverbType ( ReverbTypes type_)

sets a new reverb type

the fixed parameters for every type of reverb are defined here, they are called type parameters different to the normal parameters that can be changed by the user, the function clears the type parameters of each EarlyReflections and Decay and sets up new instances of them
