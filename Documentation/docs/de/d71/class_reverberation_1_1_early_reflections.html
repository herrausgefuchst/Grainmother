Detailed Description

handles processing of the early reflections

processes all taps of TapDelayStereo with their corresponding pan values, lowpasses allpasses and feedbacks the input

◆ processAudioSamples()

float32x2_t EarlyReflections::processAudioSamples ( const float32x2_t input_,
const unsigned int & sampleIndex_ )

processes incoming stereo samples

creates the new input for the tapdelay-buffer:

  1. alpasses the input samples
  2. adds a definable amount of feedback times the 4th early reflection in the tapdelay
  3. lowpasses the allpassed feedbacked signal

reads the taps from TapDelayStereo and multiplies them with respective pan-values, then sums up all the taps and returns a stereo float vector

Parameters
input_a vector of a pair of floats
sampleIndex_(0...blocksize) the momentary index of the audioblock sample
Returns
the processed audio samples
