Detailed Description

Represents an individual grain in granular synthesis.

The Grain class handles the lifecycle of a grain, including its envelope and data retrieval. Each grain uses a GrainProperties object to define its behavior and interacts with a SourceData object to produce audio samples.

Constructs a Grain object with the specified properties and source data.

This constructor initializes a new grain with the provided properties and creates both the GrainData and envelope objects. It also sets the grain's lifetime and marks it as alive.

float Grain::getNextSample ( )

Retrieves the next sample for the grain.

This function fetches the next audio sample for the grain by multiplying the data and the envelope values. It also decrements the life counter and marks the grain as no longer alive if its lifetime has expired.
