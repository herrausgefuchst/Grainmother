Detailed Description

A class for debouncing digital signals.

The Debouncer class is designed to filter out noise in digital input signals by implementing a time-based debounce mechanism. It transitions between stable states (OPENED and CLOSED) and intermediate states (JUSTOPENED and JUSTCLOSED) based on a configurable debounce time.
